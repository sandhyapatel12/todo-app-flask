# How to run Flask App:

1. activate virtual env                      --> env\Scripts\activate.ps1
2. python app.py


# Required:

1. create virtual env                        --> python -m venv env
2. activate virtual env                      --> env\Scripts\activate.ps1
3. deactivate your virtual environment       --> deactivate
4. create requirements.txt file              --> pip freeze > requirements.txt
5. run requirements.tsxt file                --> pip install -r requirements.txt
